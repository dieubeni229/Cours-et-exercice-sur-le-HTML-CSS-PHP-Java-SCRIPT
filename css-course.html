Introduction au CSS

Le CSS, ou Cascading Style Sheets (feuilles de style en cascade), est un langage de style qui permet de définir l'apparence des éléments HTML d'une page web. Tandis que HTML se concentre sur la structure du contenu, CSS se charge de l'aspect visuel. Le CSS est essentiel pour rendre les sites web attrayants, organisés et fonctionnels.

Les Bases du CSS

Le CSS fonctionne en ciblant les éléments HTML grâce à des sélecteurs, puis en leur appliquant des propriétés pour modifier leur apparence. Voici un exemple simple d'application de styles CSS :


/* Sélecteur de balise HTML */
h1 {
  color: blue; /* Change la couleur du texte */
  font-size: 36px; /* Définit la taille de la police */
}

p {
  line-height: 1.5; /* Modifie l'espacement des lignes */
  font-family: Arial, sans-serif; /* Définit la police */
}

Les Sélecteurs CSS

Les sélecteurs CSS sont utilisés pour cibler les éléments HTML à styliser. Voici les types les plus courants de sélecteurs :

  • Sélecteur de balise : Applique le style à tous les éléments d'une même balise. Exemple : h1 { ... }
  • Sélecteur de classe : Applique le style aux éléments ayant une classe spécifique. Exemple : .maClasse { ... }
  • Sélecteur d'identifiant : Applique le style à un élément unique avec un identifiant. Exemple : #monId { ... }
  • Sélecteur combiné : Cible des éléments qui sont dans une relation avec d'autres éléments. Exemple : div p { ... }

La Mise en Page avec CSS

Le CSS permet également de contrôler la mise en page de vos éléments HTML. Vous pouvez définir la taille, la position et l'agencement des éléments. Voici les propriétés de mise en page les plus utilisées :

  • Display : Contrôle le type de rendu d'un élément. Exemple : display: block;
  • Position : Définit la position d'un élément dans la page. Exemple : position: relative;
  • Float : Permet de faire "flotter" des éléments à gauche ou à droite. Exemple : float: left;
  • Flexbox : Permet de créer des mises en page flexibles et réactives. Exemple : display: flex;

Exemple de Mise en Page avec Flexbox

Flexbox est une méthode moderne et puissante pour gérer la mise en page. Voici un exemple simple de disposition de trois éléments :


.container {
  display: flex; /* Active le modèle Flexbox */
  justify-content: space-between; /* Espacement entre les éléments */
}

.item {
  width: 30%; /* Chaque élément occupe 30% de la largeur de son conteneur */
}

Responsivité avec CSS

Le design réactif est une approche qui permet à votre site web de s'adapter aux différentes tailles d'écran. Vous pouvez utiliser les media queries pour appliquer des styles spécifiques à différents types d'écrans (ordinateurs, tablettes, smartphones).


@media (max-width: 768px) {
  .container {
    flex-direction: column; /* Colonne sur les petits écrans */
  }
}

Conclusion du Cours CSS

Le CSS est l'un des éléments essentiels pour rendre vos pages web attrayantes et bien structurées. En maîtrisant le CSS, vous serez en mesure de créer des sites web professionnels et adaptés à tous les types de périphériques. Vous avez maintenant les bases nécessaires pour commencer à styliser vos pages web.
