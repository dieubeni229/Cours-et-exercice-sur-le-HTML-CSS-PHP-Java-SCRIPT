Introduction à PHP

PHP (Hypertext Preprocessor) est un langage de programmation côté serveur largement utilisé pour créer des sites web dynamiques et interagir avec des bases de données. Contrairement à HTML, CSS et JavaScript qui sont exécutés sur le côté client (dans le navigateur), PHP est exécuté sur le serveur avant que la page web ne soit envoyée au client.

PHP est très populaire pour la création de systèmes de gestion de contenu (CMS) comme WordPress, Joomla, et Drupal. Il permet de personnaliser le contenu des pages en fonction des besoins de l'utilisateur, de gérer des formulaires et de communiquer avec des bases de données comme MySQL.

Comment Installer PHP

Avant de commencer à écrire du code PHP, vous devez disposer d'un serveur web local pour tester vos scripts. Vous pouvez installer des logiciels comme XAMPP ou WAMP qui fournissent un environnement serveur avec Apache, MySQL, et PHP préinstallés.

Une fois installé, vous pouvez créer des fichiers PHP dans le répertoire "htdocs" (XAMPP) ou "www" (WAMP). Vous pouvez ensuite accéder à vos scripts PHP via l'URL http://localhost/monFichier.php dans votre navigateur.

Syntaxe de Base en PHP

Un script PHP est inséré dans un fichier avec l'extension .php. Vous commencez un bloc PHP avec la balise <?php et vous le terminez avec ?>.

Exemple de code PHP simple :


<?php
echo "Bonjour, bienvenue dans le cours PHP !";
?>

Dans cet exemple, la fonction echo affiche le texte à l'écran. Chaque ligne de code PHP se termine par un point-virgule (;).

Les Variables en PHP

Les variables en PHP sont utilisées pour stocker des données. Une variable commence toujours par un signe dollar ($) suivi du nom de la variable. Les variables PHP sont dynamiques, ce qui signifie que vous pouvez stocker n'importe quel type de données (texte, nombres, tableaux, objets, etc.) dans une variable sans déclaration préalable du type de données.

Exemple de variable en PHP :


<?php
$message = "Bonjour le monde !"; // Déclaration d'une variable
echo $message; // Affichage de la variable
?>

Dans cet exemple, la variable $message contient une chaîne de texte et la fonction echo affiche cette valeur sur la page web.

Les Types de Données en PHP

PHP prend en charge plusieurs types de données, parmi lesquels :

  • String : Une chaîne de caractères (ex. : "Bonjour, PHP")
  • Integer : Un entier (ex. : 42)
  • Float : Un nombre à virgule flottante (ex. : 3.14)
  • Boolean : Un type logique (ex. : true ou false)
  • Array : Un tableau ou liste de valeurs
  • Object : Une instance d'une classe (orienté objet)
  • NULL : Une variable sans valeur définie

Les Conditions en PHP

Les conditions permettent d'exécuter un code uniquement si certaines conditions sont remplies. En PHP, vous utilisez les structures conditionnelles comme if, else, et elseif.

Exemple de condition en PHP :


<?php
$age = 20;

if ($age >= 18) {
  echo "Vous êtes majeur.";
} else {
  echo "Vous êtes mineur.";
}
?>

Dans cet exemple, si la variable $age est supérieure ou égale à 18, le message "Vous êtes majeur." sera affiché. Sinon, le message "Vous êtes mineur." s'affichera.

Les Boucles en PHP

Les boucles permettent de répéter une action plusieurs fois. Les types de boucles les plus courants en PHP sont while, for, et foreach.

Exemple de boucle for en PHP :


<?php
for ($i = 0; $i < 5; $i++) {
  echo "Numéro " . $i . "<br>";
}
?>

Dans cet exemple, la boucle for affichera les numéros de 0 à 4.

Les Formulaires en PHP

PHP est couramment utilisé pour traiter les données soumises via des formulaires HTML. Vous pouvez récupérer les données d'un formulaire en utilisant les superglobales $_GET ou $_POST, selon le type de méthode utilisée dans le formulaire.

Exemple de traitement de formulaire en PHP :


<form method="POST" action="traiter.php">
Nom : <input type="text" name="nom"><br>
Prénom : <input type="text" name="prenom"><br>
<input type="submit" value="Envoyer">
</form>

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
  $nom = $_POST["nom"];
  $prenom = $_POST["prenom"];
  echo "Bonjour, " . $prenom . " " . $nom;
}
?>

Dans cet exemple, les données soumises par le formulaire sont récupérées avec la superglobale $_POST et affichées à l'utilisateur après le traitement du formulaire.

Interaction avec une Base de Données MySQL en PHP

PHP est souvent utilisé pour interagir avec des bases de données afin de stocker, récupérer et manipuler des informations. Voici un exemple de connexion à une base de données MySQL et d'exécution d'une requête SQL :


<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "ma_base_de_donnees";

// Connexion à la base de données
$conn = new mysqli($servername, $username, $password, $dbname);

// Vérifier la connexion
if ($conn->connect_error) {
  die("Échec de la connexion : " . $conn->connect_error);
}

// Exécution d'une requête SQL
$sql = "SELECT id, nom FROM utilisateurs";
$result = $conn->query($sql);

if ($result->num_rows > 0) {
  // Afficher les résultats
  while($row = $result->fetch_assoc()) {
    echo "id: " . $row["id"]. " - Nom: " . $row["nom"]. "<br>";
  }
} else {
  echo "Aucun résultat";
}

// Fermer la connexion
$conn->close();
?>

Conclusion du Cours PHP

PHP est un langage puissant pour le développement de sites web dynamiques, l'interaction avec des bases de données et la gestion des formulaires. En maîtrisant PHP, vous pourrez créer des applications web plus complexes et interactives.
