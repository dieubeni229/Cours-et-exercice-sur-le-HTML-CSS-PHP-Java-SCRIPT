Introduction à JavaScript

JavaScript est un langage de programmation puissant utilisé pour créer des pages web interactives. Il permet de manipuler des éléments HTML et CSS à la volée, de gérer des événements utilisateur, et de créer des animations et des effets dynamiques. En tant que langage côté client, JavaScript s'exécute directement dans le navigateur de l'utilisateur.

Les Bases du JavaScript

Voici un exemple basique de JavaScript pour afficher une alerte :


alert("Bienvenue sur mon site !");

Manipuler le DOM avec JavaScript

Le DOM (Document Object Model) est l'interface qui permet à JavaScript de manipuler les éléments HTML. Voici un exemple où nous modifions le contenu d'un élément HTML :


document.getElementById("monTitre").innerHTML = "Titre modifié !";

Les Événements JavaScript

Les événements en JavaScript vous permettent de réagir aux actions de l'utilisateur, comme un clic, une soumission de formulaire, ou une saisie dans un champ de texte. Voici un exemple d'ajout d'un événement de clic :


document.getElementById("monBouton").addEventListener("click", function() {
  alert("Bouton cliqué !");
});

Les Fonctions et Variables en JavaScript

JavaScript vous permet de définir des fonctions pour structurer votre code et éviter la répétition. Voici un exemple de fonction qui additionne deux nombres :


function additionner(a, b) {
  return a + b;
}
console.log(additionner(5, 3)); // Affiche 8

Conclusion du Cours JavaScript

JavaScript est essentiel pour dynamiser vos sites web et rendre vos pages interactives. Il vous permet d'ajouter des fonctionnalités complexes à vos projets web, en manipulant le DOM, en écoutant les événements et en interagissant avec des données externes via des promesses et des API.
